    GalerkinEtena.Legendre — Method.

    Legendre(x::Array{Float64}, n::Int) computes the matrices

    \[P_{ij} = P^j(x_i)\]

    and

    \[P'_{ij} = \frac{dP^j}{dx}(x_i)\]

    where

    \[P^n(x) = \left(\frac{2^nn!}\right)\frac{d^n}{dx^n}\!\left[(x^2-1)^n\right]\]

    GalerkinEtena.vander — Function.

    vander(x, n) computes the Vandermonde matrix V(x₁, x₂, ... , xₚ) defined by

    \[V_{ij} = [x_j^{i-1}]_{ij} \, \forall \, i=1,\cdots,p \, j = 1,\cdots,\n\]

    optional argument n enable us to choose the number of rows
